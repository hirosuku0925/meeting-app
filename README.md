# 🐱 Scratch猫アバター・ビデオ会議システム

Scratchのキャラクターを自分の顔の代わりに動かせる、自作のビデオ会議アプリです。
プログラムでパーツを合成して、アニメーションを実現しています。

## ✨ 主な機能
- **複数キャラクターアバター**: 🦊キツネ、🐱ネコ、🐶イヌなど複数のアバターから選択可能
- **VRoid統合**: VRoid Studioで作成したアバターをインポート可能
- **顔認識アニメーション**: MediaPipeで顔を検出、自動的にアバターが表情を合わせます
- **自動口パク**: 音声なしでも、アバターが楽しそうにパクパク動きます
- **ビデオ会議**: ルーム名を入力して、友達や家族と接続できます
- **録画機能**: 画面の下にある🔴ボタンで、通話の内容を保存できます
- **チャット**: テキストでのやり取りも可能です

## 🎮 使い方
1. **カメラ・マイクの許可**: ブラウザの許可ボタンを押してください
2. **アバター選択**: 🎭ボタンを押して、使用するアバターを選択します
3. **ルーム参加**: 好きな言葉（例：`my-room`）を入れて「参加」を押します
4. **ビデオ会議**: 他の参加者と顔の代わりにアバターで会話します

## 🛠 使用している技術
- **Vite + TypeScript**: 高速な開発環境
- **PeerJS**: サーバーなしで直接通信（P2P）を実現
- **Three.js**: 3D アバター表示
- **MediaPipe**: 顔認識と表情分析
- **VRM フォーマット**: VRoid との互換性

### アバター追加方法
詳細は `public/AVATARS_SETUP.md` を参照してください。

VRoid Studio で作成したモデルは、VRM 形式でエクスポートして `public/avatars/` に配置することで、アプリから利用できます。
