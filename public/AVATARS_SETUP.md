# アバター機能の設定ガイド

このアプリケーションでは、複数のキャラクターアバターから選択できます。

## 利用可能なアバター

現在、以下のアバターが利用可能です：

- 🦊 **キツネ** - かわいいキツネアバター
- 🐱 **ネコ** - かわいいネコアバター
- 🐶 **イヌ** - かわいいイヌアバター
- 🐰 **ウサギ** - かわいいウサギアバター
- 🐻 **クマ** - かわいいクマアバター
- 🤖 **ロボット** - クールなロボットアバター
- 👽 **エイリアン** - ユニークなエイリアンアバター
- 👤 **デフォルト** - デフォルトアバター

## アバターモデルの追加方法

### 1. VRMモデルの準備

VRoid Studio または他のツールで VRM 形式のモデルを作成します：

1. VRoid Studio を開く
2. アバターを作成または編集
3. **メニュー → ファイル → エクスポート** を選択
4. **VRM 形式** でエクスポート

### 2. ファイルの配置

ダウンロードした VRM ファイルを `public/avatars/` ディレクトリに配置します：

```
my-meeting-app/
  ├── public/
  │   ├── avatars/
  │   │   ├── fox.vrm           # キツネアバター
  │   │   ├── cat.vrm           # ネコアバター
  │   │   ├── dog.vrm           # イヌアバター
  │   │   ├── rabbit.vrm        # ウサギアバター
  │   │   ├── bear.vrm          # クマアバター
  │   │   ├── robot.vrm         # ロボットアバター
  │   │   ├── alien.vrm         # エイリアンアバター
  │   │   └── ...
  │   └── vroid-avatar.vrm      # デフォルトアバター
  └── ...
```

### 3. avatar-manager.ts の設定

新しいアバターを追加する場合、`src/avatar-manager.ts` の `AVATARS` 配列に追加します：

```typescript
{
  id: 'your-avatar-id',
  name: 'アバター名',
  emoji: '😀',
  description: 'アバターの説明',
  modelPath: '/avatars/your-avatar.vrm'
}
```

## アバター機能の使い方

### 1. アバター選択

1. UI の下部ツールバーで **🎭 アバター** ボタンをクリック
2. アバター選択ダイアログが表示される
3. 希望のアバターをクリック
4. アバタープレビューが大画面に表示される

### 2. アバター表示

- アバターが選択されると、3D プレビューが表示されます
- アバターは自動的に回転して表示されます
- 「閉じる」ボタンで終了

## トラブルシューティング

### VRM ファイルが見つからない

**原因**: ファイルパスが正しくない、またはファイルが配置されていない

**解決策**:
1. ファイルが `public/avatars/` に配置されているか確認
2. ファイル名と拡張子が正確か確認
3. avatar-manager.ts のパスが正しいか確認
4. ブラウザコンソール (F12) でエラーを確認

### アバターが表示されない

**原因**: VRM モデルの形式が不正、または Three.js ライブラリが正しく読み込まれていない

**解決策**:
1. VRM ファイルが有効な形式か確認
2. ブラウザコンソールでエラーを確認
3. 別の VRM ファイルを試す

## カスタマイズ

### 表情アニメーション

`src/vrm-face-sync.ts` で顔認識と VRM の表情マッピングを調整できます。

### アバター選択ダイアログの外観

`src/avatar-dialog.ts` でダイアログのスタイルをカスタマイズできます。

## パフォーマンス

複数のアバターをロードする場合、以下に注意してください：

- 各 VRM ファイルは複数のテクスチャを含む可能性があります
- ファイルサイズが大きい場合、ロード時間が長くなります
- 不要なアバターは削除することでメモリ使用量を削減できます

## サポート

トラブルが発生した場合は、ブラウザコンソール (F12) のエラーメッセージを確認してください。
